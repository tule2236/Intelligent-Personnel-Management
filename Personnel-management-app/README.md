# Personnel-management-app

This application is a cross-platform desktop app intended to be used by small teams for collaboration. Please see the instructions below to get started.

## Prerequisites

You will need the following things properly installed on your computer.

* [Git](http://git-scm.com/)
* [Node.js](http://nodejs.org/) (with NPM)
* [Bower](http://bower.io/)
* [Ember CLI](http://ember-cli.com/)
* [PhantomJS](http://phantomjs.org/)

## Installation

* `git clone <repository-url>` this repository
* `cd personnel-management-app`
* `npm install`
* `bower install`

## Running / Development

* `ember electron`
* An electron window should be generated
